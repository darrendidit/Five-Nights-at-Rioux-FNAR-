# Five-Nights-at-Rioux-FNAR-

## Description
fnaf inspired game based off of my computer science teacher and my chemistry teacher's never ending beef

## How To Play
1. Press New Game
2. Press Start Night
3. Look at the cameras, and check/close the door to survive from Ms. Rioux, Chenjie, Ansh, and Joseph (Former CSHS President)
4. Survive for 3 minutes and you'll beat the night.
5. Repeat steps 2-4 for nights 2-5 and you'll win the game! (no winscreen tho)

**if you die, you lose all of your progress and have to restart. have fun!

## Special Features
1. Usage of Java Swing
   - helps provide visuals to the base java
2. Usage of a timer
   - gives the user pressure to "complete the night" and "lock in"
3. Camera System
   - being able to watch your surroundings really sets the tone that you are stuck in the office and need to survive
4. Usage of Images
   - helps provide even more visuals and grants the user better visuals that I could make myself
