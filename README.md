# Five-Nights-at-Rioux-FNAR-

## Description
fnaf inspired game based off of my computer science teacher and my chemistry teacher's never ending beef

## How To Play
1. Press New Game
2. Press Start Night
3. Look at the cameras, and check/close the door to survive from Ms. Rioux, Chenjie, Ansh, and Joseph (Former CSHS President)
4. Survive for 3 minutes and you'll beat the night.
5. Repeat steps 2-4 for nights 2-5 and you'll win the game! (no winscreen tho)
**if you die, you lose all of your progress and have to restart. have fun!
